🏋️ Embedded AI Fitness Assistant

라즈베리파이 기반 임베디드 시스템에서 실시간 포즈 추정과 PID 제어를 결합한 운동 보조 시스템입니다.
YOLOv8을 활용해 스쿼트·벤치프레스·데드리프트 자세를 분석하고, 카메라를 서보모터로 제어하여 사용자를 추종하도록 구현했습니다.

🔧 Tech Stack

Raspberry Pi (Linux)

Python, OpenCV

YOLOv8 Pose

PID Control

Servo Motor Control

🚀 Key Points

실시간 포즈 추정 및 각도 기반 자세 분석 알고리즘 구현

PID 제어 기반 카메라 추종 시스템 설계

영상 처리와 모터 제어를 통합한 임베디드 시스템 구현
